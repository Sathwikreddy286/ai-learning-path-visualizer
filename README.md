# AI Learning Path Visualizer

## Overview

AI-based web app that generates a **structured learning roadmap** for any skill or career goal.

---
## Problem

Learners often face confusion due to:
* No clear roadmap
* Too many scattered resources
* Not knowing what to learn next

## Solution

This app uses AI to:
* Take user goal as input
* Generate a **step-by-step learning path**
* Organize it into Beginner → Intermediate → Advanced

## Features

* AI-generated learning paths
* Clear and structured roadmap
* Simple and user-friendly interface
